# ☀️ SolAura - Sunscreen Product Website

![SolAura Banner](https://img.shields.io/badge/SPF-50+-orange) ![License](https://img.shields.io/badge/license-MIT-green) ![Docker](https://img.shields.io/badge/Docker-ready-blue) ![Nginx](https://img.shields.io/badge/Nginx-alpine-brightgreen)

**SolAura** est un site web vitrine one-page pour un produit de protection solaire imaginaire. Design épuré, responsive, et prêt à être déployé avec Docker et Nginx.

![Aperçu du site](https://via.placeholder.com/800x400/fefaf5/d4943a?text=SolAura+Sunscreen+Preview)

---

## 📸 Aperçu rapide

Le site présente :
- 🧴 Un hero section avec appel à l'action
- 🌿 Une grille d'ingrédients naturels (Zinc, Niacinamide, Squalane, Bêta-Carotène)
- 💬 Un témoignage client intégré
- 📱 Un design entièrement responsive (mobile, tablette, desktop)
- 🎨 Des dégradés chauds évoquant le soleil et la nature

## 🚀 Installation rapide

### Prérequis
- **Option 1** : Un navigateur web (ouvrir `index.html`)
- **Option 2** : [Docker](https://docs.docker.com/get-docker/)
- **Option 3** : [Nginx](https://nginx.org/) (optionnel)

### Méthode 1 : Ouverture directe (la plus simple)
```bash
git clone https://github.com/VOTRE-USERNAME/solaura-sunscreen.git
cd solaura-sunscreen
# Ouvrir index.html dans votre navigateur
firefox index.html   # ou chrome, brave, etc.
